# gahoerning.github.io

Personal academic website of **Gabriel A. Hoerning**, PhD student in Astronomy and Astrophysics at the University of Manchester and the Jodrell Bank Centre for Astrophysics.

The website is hosted with **GitHub Pages** at:

<https://gahoerning.github.io>

## Website structure

The site currently includes:

- `index.html` — homepage
- `research.html` — research interests and current projects
- `products.html` — scientific data products, software tools, and documentation
- `publications.html` — publication links and selected papers
- `code.html` — selected repositories and analysis tools
- `cv.html` — CV page
- `assets/style.css` — website styling, including dark/light mode

## Scope

This website is intended to serve as a personal academic hub, including:

- research overview
- scientific products and data-release links
- selected code repositories
- publications
- CV
- contact and profile links

## Technical notes

The site is built using plain HTML, CSS, and client-side JavaScript. It is statically hosted through GitHub Pages.

The default visual theme is dark mode, with a light/dark toggle implemented using JavaScript and `localStorage`.

## Large files

Large scientific products, such as FITS maps, data cubes, chains, or full data releases, should not be hosted directly in this repository.

Preferred locations for large files are:

- Zenodo
- institutional repositories
- project data archives
- external cloud/data services

This repository should mainly contain the website source code, small assets, documentation, and links to external data products.
