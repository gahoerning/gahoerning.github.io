Code

This page lists selected repositories, scripts, and analysis tools associated with my research.

My GitHub profile is available here: github.com/gahoerning.

Selected repositories

  • Commander analysis tools
    Scripts for preparing sky maps, applying masks, smoothing data, producing residual maps, and generating diagnostic plots.
  • RRL and free-free analysis
    Tools and notebooks for analysing hydrogen radio recombination line data and comparing RRL-derived free-free emission with radio continuum maps.
  • AME residual analysis
    Scripts for estimating anomalous microwave emission residuals in Galactic star-forming regions.
